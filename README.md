# nano-banana-basic-python

## Preparation
- `uv sync`
- Rename `.env.sample` to `.env`
- Generate api key at Google AI Studio (api key must be linked to a Google Cloud project associated with a billing account)
- Copy and paste generated api key into `.env` file
- Set your image output directory into `.env` file

## Usage
`uv run --env-file .env main.py "Funny Cat Please"`

##  Example Prompts
- 基本的な８ビートをイメージさせる画像を作ってね。ドラムセットと👄マークと👃マークが入り乱れる感じのイメージです。「basic-8th-note-groove」という文字を左上に入れてください。また、右下にはウォーターマークセクション（AIで作成されたことが分かる文字列およびブログ投稿者の手書き承認文字が後で入るところ）があり、まず「Generated by AI」という文字を入れ、その真下に「Verified by」という文字を入れてください（これは後でブログ投稿者が画像編集アプリ上でマニュアルにてシグネチャーを入れるためのものです笑）
- 「HUM 'em ALL」というダサかっこいい小さめのロゴを作ります。ロゴ自体は全体キャンバスの中央付近に寄せて描いてください。ロゴはロックバンド的な形のニュアンスです。👄マークと👃マークがロゴのそれぞれ右左に置かれており、口の中から鼻の穴までクラウドネットワークを想起させるような配線が引かれている感じでお願いします。
- コロナ禍の状況において、マンションの中に閉じ込められた人々が気晴らしで窓から口を出して鼻歌を歌っている画像を作ってください。マンションはそれぞれ離れた棟が描かれており、その離れた棟の窓同士でなんとかコミュニケーションを取り合う感じのアニメ風の画像でよろしく。「HUM 'em ALL」という赤系のロゴを中央にいれて、その少し上方に「Harmony in Lockdown」という文字を違う色で虹がかかるようにいれてください。また、マンションの上に5Gのアンテナをたててください。また、右下にはウォーターマークセクション（AIで作成されたことが分かる文字列およびブログ投稿者の手書き承認文字が後で入るところ）があり、まず「Pic Generated by AI」という文字を入れ、その真下に「Approved by Human」という文字を入れてください。Humanという文字は少し薄く表示します（この上に重ねて人間がシグネチャーを手書きで入れるため）
